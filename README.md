# App Itinerary Thailand

A web application for planning and managing travel itineraries in Thailand.

## Features

- Create and customize trip itineraries
- Plan daily activities and destinations
- Track locations and travel times
- Responsive design for mobile and desktop

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Vite
